# Nodly

Візуальний калькулятор на основі вузлів (node-based calculator) для мобільних пристроїв та браузера.

## Що це

Nodly — це застосунок де обчислення будуються як граф:
**Число → Формула → Результат**

Подобається тим, хто хоче бачити як пов'язані величини між собою — для інженерних розрахунків, фізики, фінансів, будівництва.

## Можливості

- **Вузли**: Число, Формула, Результат, Нотатка
- **Формули**: `+`, `-`, `*`, `/`, `**`, `sqrt()`, `sin()`, `cos()`, `log()`, `round()`, `min()`, `max()`, `pi`, `e` та інші
- **З'єднання**: з'єднуй вихід одного вузла зі входом іншого
- **Фото**: додавай зображення або знімай з камери прямо на вузол
- **Масштаб вузлів**: S / M / L / XL
- **Зміна ширини**: тягни ричажок у правому нижньому куті
- **Авто-впорядкування**: розставляє ноди за зв'язками зліва-направо
- **Збереження**: автоматично у браузері (IndexedDB)
- **Кілька калькуляторів**: зберігай і дублюй
- **Експорт**: PNG-знімок полотна
- **Кирилиця**: змінні у формулах можна писати українськими буквами

## Як запустити

Відкрий `index.html` у будь-якому сучасному браузері. Не потребує сервера.

## Технологія

Чистий HTML + CSS + Vanilla JavaScript. Без фреймворків і залежностей.
